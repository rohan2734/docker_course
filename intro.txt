* Docker : is a platform or ecosystem around creating and running containers
* Image: single file with all dependencies and config required to run a program
* container: Instance of an image. we can think of it as beng like a running program
  it is a program with its own isolated set of hardware resources
  it has its own little space of memory,netowrking technology, and hardware resources

* DockerCLI (Dockerclinet) is a tool or portal to help us interact with other software called DockerServer(Docker Daemon)
* Docker Server is sresponsible for creating containers, images, maintaining containers and uploading images.
